# Thesis Project Structure

- paper/             LaTeX 主文档与编译相关文件
- paper/chapters/    章节内容（每章一个 .tex）
- paper/figures/     论文图片与图表
- paper/tables/      论文表格（导出或手工维护）
- paper/appendix/    附录内容
- refs/              BibTeX 文献库（.bib）
- code/              实验代码与脚本
- data/              数据目录
- data/raw/          原始数据
- data/processed/    处理后的数据
- results/           实验输出、日志、模型、指标等
- notes/             研究/写作过程中的笔记
